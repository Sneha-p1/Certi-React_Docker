# CertiApp
CertiApp is a web application for managing and displaying certificates with student details.
## Features
- Create and manage certificates
- View student details
- Responsive design
## Technologies
- Frontend: React, Vite
- Backend: Express.js
- Database: MongoDB
- Docker
## Installation
## Prerequisites
- Node.js (>= 14.x)
- npm or Yarn
- Docker (optional)
